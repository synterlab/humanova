# Humanova

**DIY Toolkit Platform** for creative industry practitioners and NGOs.

🌐 Live: [humanova.run](https://humanova.run)

## What is Humanova?

Humanova is a curated library of 30+ practical toolkits grounded in innovation, design thinking, and social development. Browse, use, and submit toolkits to help practitioners invent, adopt, and adapt ideas that deliver better results.

## Features

- 30 toolkits across 8 intent categories
- Mobile-first, fully responsive design
- Search and filter by category, difficulty, and format
- Submit toolkit wizard (5-step)
- User authentication UI (Supabase-ready)
- Admin panel
- Community leaderboard
- Case studies

## Tech Stack

Single-file HTML/CSS/JS — zero build step, instant Vercel deployment.

Ready for upgrade to Next.js 14 + Supabase + Tailwind CSS.
